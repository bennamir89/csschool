2. Explain: The For Loop

Syntax Structure

In Python, a count-controlled loop uses the for keyword combined with range().

for i in range(5):
    print("Hello")

Key Terms:
关键词 / 핵심 용어

  • for: Keyword to start the loop. 关键字：开始循环
  • i: The iterator (variable) that changes each time. 迭代器（变量）
  • range(): A function that generates a sequence of numbers. 生成数字序列的函数

How `range()` works

range(stop) generates numbers from 0 up to (but not including) stop.

Code Output Explanation
range(5) 0, 1, 2, 3, 4 Stops before 5. (5 numbers total)
range(1, 4) 1, 2, 3 Start at 1, stop before 4.

2. Explain: Tracing the Loop

Let's trace exactly what happens when this code runs.

for count in range(3):
    print("Number is", count)

Iteration Loop # Value of 'count' Action
1st 0 Prints "Number is 0"
2nd 1 Prints "Number is 1"
3rd 2 Prints "Number is 2"
4th (Stop) Loop ends because 3 is not < 3.
